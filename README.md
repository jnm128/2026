# Philippines - Country Profile

A web page providing an overview of the Philippines, covering its history, geography, people, and environment.

## Project Structure

- `Test.html` — Main HTML page
- `css/style.css` — Stylesheet
- `images/` — Image assets (flag, country photo)

## Contents

- **Introduction** — Historical background from Spanish colonization to present day
- **Geography** — Location, land use breakdown, and natural resources
- **People and Society** — Ethnic groups, religions, and demographic profile
- **Environment** — Current issues, international agreements, and climate
